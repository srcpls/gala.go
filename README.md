welcome to gala.go!!
~~~~~~~~~~~~~~~~~~~
gala.go is a project I've been working on to (hopefully) eventually serve as a sort of playing field for a terrible go playing program called gala.go.  

features as of now
  -stone grouping
  -grouping of empty space(which should help in determining territory if playing by japanese rules)
  -group capturing
  
  
currently working on

  -ko rule
  -simple gui
